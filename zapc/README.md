# ZAP code generation for Rust

[![crates.io](https://img.shields.io/crates/v/zapc.svg)](https://crates.io/crates/zapc)

[documentation](https://docs.rs/zapc/)

The generated code depends on the [zap-rust runtime library](https://github.com/zap/zap-rust).

Code generation can be customized through the annotations defined in [`rust.zap`](rust.zap).
